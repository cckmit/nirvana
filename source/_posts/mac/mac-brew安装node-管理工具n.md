## 安装
```
brew install n
```
## 安装/激活版本
只需执行n <version>即可安装node。如果<version>已经安装（通过n），n将激活该版本。
例如：
```
   $ n 10.0.1
   $ n 16.9.1
```
n单独执行是查看当前安装的版本，使用向上和向下箭头键导航并按enter或右箭头键进行选择，使用^ C（控制+ C）退出选择屏幕。
```
$ n
 
  10.8.14
ο 10.8.17
  10.9.6
```
